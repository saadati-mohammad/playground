# NgRx Learning Project

This project is a hands-on NgRx playground built with Angular 19.
The goal is to deeply understand NgRx concepts such as:

- Store
- Actions
- Reducers
- Effects
- Selectors
- Entity Adapter

## Feature
- Product feature with mocked API
- Real-world folder structure
- Redux DevTools enabled

## Run
```bash
npm install
npm start
